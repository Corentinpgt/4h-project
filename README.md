Le but de ce projet était de voir ce que je pouvais produire dans un court laps de temps.

Sujet : Développer une application de livre en ligne.
Objectifs :

- Un /admin avec des controller CRUD pour chaque entity

Technologie : Symfony
Durée : 4h
